üì¢ **Ansi-To-Html Twig Extension** is a [Twig](https://twig.symfony.com) extension to convert ANSI escapes (terminal formatting/color codes) to HTML:

It is based on [ANSI escapes to Html](https://github.com/neilime/ansi-escapes-to-html).

```twig
{% ansitohtml %}
Default \e[34mBlue
{% endansitohtml %}
```
